# 毛概刷题站

一个可本地直接使用的毛概刷题网站，已整合 9 份题库 PDF 和答案文档。

打开方式：

1. 直接双击 `index.html`
2. 如果浏览器拦截本地脚本，再在当前目录运行：

```bash
python3 -m http.server 4173
```

然后访问 `http://127.0.0.1:4173`

功能说明：

- 支持按单元刷、全库混刷、只刷错题
- 支持单选、多选、判断分开练
- 支持题目乱序、选项乱序
- 已整合标准答案与解析，支持自动判分
- 自动保存当前进度、错题本、补录答案到浏览器本地

注意：

- 题库总数为 462 题
- 做题记录保存在当前浏览器本地存储中
- 如果更新过题库数据后页面显示异常，可以先点“清空当前进度”再重新开始
